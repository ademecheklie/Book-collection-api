Welcome to the Books Collection API

This API allows you to manage a collection of books with features like CRUD operations, favoriting, and personalized recommendations.

Base URL

https://book-collection-api-one.vercel.app/

Swagger Documentation

Access the interactive API documentation:

  • Swagger UI
  • Swagger JSON

Endpoints

  • GET /books/all: Fetch all books
  • POST /books: Add a new book
  • PUT /books/:id: Update a book
  • DELETE /books/:id: Delete a book
  • PATCH /books/:id/favorite: Toggle the favorite status of a book
  • GET /books/recommended: Get a recommended book
  • GET /books/favorites: Get all books marked as favorites by the user

Authentication

This API uses JWT-based authentication. Include your token in the Authorization header as a Bearer token for protected routes:

  Authorization: Bearer <your-token>

Features

  • Secure API with JWT authentication
  • CRUD operations for managing books
  • Personalized recommendations
  • Track favorite books
