Welcome to the Books Collection API

This API allows you to manage a collection of books with features like CRUD operations, favoriting, and personalized recommendations.

Base URL

https://book-collection-api-one.vercel.app/

Access the interactive API documentation:

Swagger UI

Endpoints

Authentication

  • POST /auth/signup - Sign up a new user
  • POST /auth/login - Log in a user

Books

  • GET /books - Get all books (requires authentication)
  • POST /books - Create a new book (requires authentication and user role)
  • PUT /books/:id - Update a book by ID (requires authentication and user role)
  • DELETE /books/:id - Delete a book by ID (requires authentication and user/admin role)
  • PATCH /books/:id/favorite - Toggle favorite status of a book (requires authentication and user role)
  • GET /books/recommended - Get a recommended book (requires authentication)
  • GET /books/favorite - Get all favorite books (requires authentication)
  • GET /books/:id - Get a book by ID (requires authentication)

Models

User

  • id - The auto-generated ID of the user
  • name - The name of the user
  • username - The username of the user
  • email - The email of the user
  • password - The password of the user
  • role - The role of the user (admin/user)

Book

  • id - The auto-generated ID of the book
  • title - The title of the book
  • author - The author of the book
  • isbn - The ISBN of the book
  • publishedYear - The published year of the book
  • createdBy - The ID of the user who created the book
  • isFavorite - The favorite status of the book

Security

This API uses JWT for authentication. Include the token in the Authorization header as Bearer <token>.
