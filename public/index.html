Welcome to the Books collection API

This API allows you to manage a collection of books with features like CRUD operations, favoriting, and recommendations.

Base URL

https://book-collection-api-one.vercel.app/

Endpoints

  • GET /books: Fetch all books
  • POST /books: Add a new book
  • PUT /books/:id: Update a book
  • DELETE /books/:id: Delete a book
  • PATCH /books/:id/favorite: Toggle favorite status
  • GET /books/recommended: Get a recommended book
